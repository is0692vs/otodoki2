# Videos

このディレクトリには、otodoki2 アプリケーションのデモ動画やアニメーション GIF を配置します。

## 配置するファイル

- `auth-flow-demo.mp4` / `auth-flow-demo.gif` - 登録・ログインフローのデモ
- `swipe-demo.mp4` / `swipe-demo.gif` - スワイプ機能のデモ
- `library-demo.mp4` / `library-demo.gif` - ライブラリ機能のデモ
- `audio-preview-demo.mp4` / `audio-preview-demo.gif` - オーディオプレビュー機能のデモ
- `mobile-app-demo.mp4` / `mobile-app-demo.gif` - モバイルアプリのデモ
- `setup-demo.mp4` / `setup-demo.gif` - セットアップ手順のデモ

## ガイドライン

### MP4 動画

- **コーデック**: H.264 推奨
- **最大サイズ**: 10MB
- **推奨解像度**: 1280x720 以下
- **フレームレート**: 15-30 fps
- **長さ**: 30-60 秒推奨

### GIF アニメーション

- **最大サイズ**: 5MB
- **推奨解像度**: 800x600 以下
- **フレームレート**: 10-15 fps
- **長さ**: 10-30 秒推奨

### 命名規則

- `機能名-demo.mp4` または `機能名-demo.gif` (小文字、ハイフン区切り)

## 動画作成のヒント

1. **画面録画**: OBS Studio, QuickTime, Loom などを使用
2. **編集**: 不要な部分をカットして短く保つ
3. **GIF 変換**: FFmpeg や ezgif.com を使用して最適化
4. **サイズ削減**: 解像度やフレームレートを調整

詳細は [VISUAL_GUIDE.md](../VISUAL_GUIDE.md#メディアファイルの追加方法) を参照してください。
